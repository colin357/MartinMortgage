# Image Upload Instructions

Drop your images into this folder (`/public/images/`) and they will appear on the website automatically.

## Required Images

| Filename | Where it appears | Recommended size |
|---|---|---|
| `logo.png` | Header (top navigation) — replaces text logo | 480×128 px, transparent PNG |
| `michael.png` | Homepage hero (right side, full body) | 600×800 px, transparent PNG background preferred |
| `michael-headshot.png` | About section card (circular crop) | 400×400 px |

## Tips
- Use PNG with a transparent background for `michael.png` so it blends with the white hero section
- The logo slot is 240px wide × 64px tall max
- Images are served from `https://yoursite.com/images/filename.png`
- After adding images, redeploy the site on Vercel (or it will auto-deploy if you push to GitHub)
